# 🏨 Hotel Reservation System (Java)

A console-based **Hotel Reservation System** developed in **Java** using **Object-Oriented Programming (OOP)** principles and **File Handling**. This application allows users to search, book, cancel, and manage hotel room reservations while storing booking details in a text file.

---

## 📌 Features

- 🔍 Search available hotel rooms
- 🛏️ Book rooms by room number
- ❌ Cancel existing reservations
- 📋 View all current bookings
- 🏷️ Room categorization:
  - Standard
  - Deluxe
  - Suite
- 💳 Payment simulation
- 💾 Store booking details using File I/O (`bookings.txt`)
- 🖥️ Console-based user interface
- 📚 Object-Oriented Programming implementation

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/HotelReservationSystem.git
```

### 2. Open the project

Open the project folder in **VS Code** or any Java IDE.

### 3. Compile the project

```bash
javac *.java
```

### 4. Run the application

```bash
java Main
```

## 💳 Payment Simulation

The application simulates different payment methods:

- UPI
- Credit Card
- Cash

After successful payment, the booking is confirmed and stored in `bookings.txt`.

---

## 📄 Booking Storage

Bookings are automatically saved in:

```
bookings.txt
```

Example:

```
Rahul,101,Standard
Aman,201,Deluxe
Saurabh,301,Suite
```
